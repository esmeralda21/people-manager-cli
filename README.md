# People Manager CLI

Add, modify, delete, and display information from a user database

## Features
- Add a new user
- View the complete database
- Search for a user by name
- Modify a user's field
- Delete a user

## Requirements
Python 3.12+

## How to run

1. Clone the repository: `git clone https://github.com/esmeralda21/people-manager-cli.git`
2. Navigate to the folder: `cd people-manager-cli`
3. Run the program: `python main.py`

## Project structure
`main.py` — menu logic and database operations
`funciones.py` — validation and utility functions

## What I learned
CRUD operations with SQLite
Documenting functions with docstrings
Input validation within a CLI menu
